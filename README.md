Fungsi:
Menentukan bilangan yang dimasukkan pengguna apakah ganjil atau genap
Cara Kerja:
Program meminta pengguna untuk memasukkan sebuah bilangan bulat
Program menggunakan operator modulus (%) agar menghitung sisa hasil bagi bilangan tersebut dengan 2
Jika sisa bagi adalah 0 (angka % 2 == 0), berarti bilangan tersebut termasuk bilangan genap
Jika sisa bukan 0, maka bilangan tersebut adalah bilangan ganjil
Program mencetak hasil penentuan (ganjil atau genap) ke layar output
